## 🚀 Kurulum Talimatları

1. **İndir:** Depoyu ZIP olarak indirin.
2. **Kodi'yi Açın:** Ana menüden `Eklentiler` sekmesine gidin.
3. **Yükle:** `Zip dosyasından yükle` seçeneğini seçin ve indirdiğiniz dosyayı gösterin.
4. **Yapılandır:** Eklenti ayarlarından **Xtream API** kullanıcı bilgilerinizi tanımlayın.
